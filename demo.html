Here shows the description of queries and their SQL statements we are going to demo.

  1. Subquery
    1. List all customers who paid more than the average payment of all the customers in database order by inv_id.
      2. SQL Statement 
                                                  
    SELECT * FROM Invoice WHERE AmountUSD
    >(SELECT AVG(AmountUSD) FROM Invoice)
    ORDER BY inv_id;
  2. Join Table Queries
    1. Retrieve info about a customer named JSTOR, including the name and payment date.
    2. Retrive info about each customer, including cus_id, payment_ date, amountUSD and order by their cus_id.
      3. SQL Statement 
                                                  
   SELECT name, payment_date 
   FROM customer 
   JOIN Invoice ON Customer.cus_id=Invoice.cus_id 
   WHERE name='JSTOR';
      SQL Statement 
                                                  
   SELECT Invoice.cus_id, Invoice.payment_date, Invoice.amountUSD 
   FROM Invoice 
   JOIN Customer ON Customer.cus_id=Invoice.cus_id 
   ORDER BY Customer.cus_id;
  3. Group By Query
    1. Finding out how many customers have the same contact person, grouped first by contact person, and then by customer name. Also order first by contact person then by customer name.
      2. SQL Statement 
                                                  
    SELECT COUNT(*), name, contact_person 
    FROM Customer 
    GROUP BY contact_person, name 
    ORDER BY contact_person, name;
  4. Other Query
    1. Create a view called Payment that includes cus_id and the amountUSD renames as paymentamount for all the customers that have a paymentUSD greater than $9000.
      2. SQL Statement 
                                                  
    CREATE VIEW Payment AS select cus_id, amountUSD AS PaymentAmount 
    FROM INVOICE 
    WHERE amountUSD>9000;
